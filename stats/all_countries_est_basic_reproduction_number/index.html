Estimated Basic Reproduction Number: By Country

The following table contains a ranking by country ordered by the highest basic reproduction number R0 first. The definition and parameterization applied for estimating the basic reproduction number is described in the “Epidemiologischen Bulletin” of the Robert Koch Institute. In short, this bulletin makes use of the assumptions that a generation has a duration of four (4) days.

The table below skips the data of the most recent three (3) days, as the data is expected to still be too fluctuating.

Note that the value of R0 is calculated based on a time frame and not from a single day. It therefore has also an averaging effect. It does not make sense to attribute it to a single day.

Assistance for interpretation:

  • If the basic reproduction number is 1, then one infected person will also infect another person.
  • If the basic reproduction number is below 1, then one infected person (statistically) infects less than one other person. The infection rate then is assumed to be sub-linear, and the disease (slowly) peters out.
  • If the basic reproduction number is above 1, then one infected person (statistically) infects more than one other person. The infection rate then is assumed to be in expontential growth.
